The introduction and comparison between Interactive Legend and StoryMaps


Distribution of Kindergartens in Singapore


What is Interactive Legend?

The ArcGIS Online Interactive Layer is an Instant App template that offers an intuitive experience, enabling users to configure and toggle layers for their specific spatial analysis requirements. The interface mirrors the version found on ArcGIS Pro, but the features are more streamlined. Interactive Legend has an option to add a short description before the web map loads, providing users with a understanding of the data being used.

Comparison

The Interactive Legend template and StoryMaps serve distinct purposes, each offering unique features tailored to different requirements.

The Interactive Legend is designed for quick and efficient spatial analysis, allowing users to configure layers and toggle settings for their analysis requirements. With limited description and summary options, this template appears to be more suited to new users who are familiar with the data used. For example, a web map for office internal usage.

On the other hand, StoryMaps places emphasis on the narration of maps, blending maps with storytelling. Users can integrate multimedia elements, text, and interactive elements, aligning with the purpose and presentation of the maps.

What is StoryMaps?

ArcGIS StoryMaps is a dedicated web application that integrates maps with storytelling, offers users a more informative and enriched experience. By incorporating multimedia and text, representation of map data can be enhanced. The process is seamless and user-friendly. Users have the flexibility to choose from built-in style templates, customise layouts and interaction to align how the story is revealed and narrated.

Reducing Landslides at Kutupalong Refugee Camp: Description

This StoryMap raises awareness of the humanitarian crisis in the largest refugee settlement, shedding light on the dire living conditions of Rohingya refugees. The maps provide insights and pinpoints areas susceptible to landslide.
